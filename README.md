# The Flock Network
A simple (socket.io) websocket server for managing [the-flock](https://github.com/yapalexei/the-flock) client connections.

## Prerequisites
- node version: 12.12.0
- npm version: 6.11.3
- (optional) Docker

## Setup and run steps
- `yarn`
- `yarn start`
- OR (optionally) `yarn docker`
